🤖 Agentic RAG Assistant for Coding Documentation
An advanced Retrieval-Augmented Generation (RAG) project designed to query project documentation generated by Agentic Coding tools (such as Cursor, Claude Code, etc.). The system integrates Semantic Search (Vector Search) with Structured Data Extraction to provide precise answers even for quantitative and administrative queries.

🚀 Key Features
Hybrid Search: Intelligent routing between Vector Search (Pinecone/Local) and a structured database (JSON).

Event-Driven Workflow: Modern architecture built using LlamaIndex Workflows.

Self-Healing: Advanced Query Rewriting capabilities that trigger automatically if no relevant information is found.

Interactive UI: A user-friendly chat interface powered by Gradio.

🛠 Tech Stack
Framework: LlamaIndex

LLM & Embeddings: Cohere (Command-R)

Database: JSON / Vector Store

Interface: Gradio

📊 Process Architecture
🛡️ Guardrails & Infinite Loop Prevention
As shown in the architecture, the system includes a self-correction loop (from validate to rewrite_query and back to StartEvent).

Note: This is not an infinite loop. The system manages a Retry Counter limited to a maximum of 3 attempts. If the Agent fails to find relevant information after 3 rewrites, the system executes a controlled stop (via StopEvent) and informs the user that the requested information is not available in the project documentation.
